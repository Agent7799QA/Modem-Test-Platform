# Device Manager

**Document ID:** DEV-005

---

# Purpose

Device Manager управляет доступными устройствами.

---

# Responsibilities

- обнаружение устройств;
- регистрация устройств;
- подключение;
- отключение;
- поиск устройств.

---

# Rules

Device Manager является единой точкой доступа к Device.
