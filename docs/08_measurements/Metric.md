# Metric

**Document ID:** MES-002

---

# Purpose

Metric представляет вычисляемый показатель качества испытания.

---

# Examples

- LQ
- RSSI
- Packet Loss
- Reconnect Count
- Error Count

---

# Responsibilities

Metric вычисляется на основании Measurement.

---

# Rules

Набор Metric является расширяемым.
