# Events

**Document ID:** ENG-006

---

# Purpose

Все изменения состояния Test Engine оформляются в виде событий.

---

# Events

TestStarted

TestCompleted

StepStarted

StepCompleted

DeviceConnected

DeviceDisconnected

MeasurementReceived

WarningRaised

ErrorRaised

ReportGenerated

---

# Rules

События являются неизменяемыми.

События публикуются Test Engine.
